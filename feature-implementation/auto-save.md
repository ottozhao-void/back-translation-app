
# Feature Overview

用户在输入译文时，不是一气呵成的，往往断断续续，为了避免由于一些潜在的误操作，导致已输入文本消失。所以就迫切的需要系统提供自动保存的功能，当用户停止编辑一定时间，比如3秒后，或者在按下保存键后，可以是 ctrl+s，系统会自动将已输入的译文放置在一个缓冲区中，这样在用户进行某些操作时，比如刷新页面，系统会提示用户是否保存当前译文草稿。

用户可以在一个单独的设置界面，设置是否开启自动保存功能，以及被视为需要自动保存的停止编辑时间


# UI/UX

- 打开设置界面的图标应该放置在右上角，暗/亮主题图标的右边
- 自动保存的状态应该在Your Translation卡片的左下角实时更新

# Design

对于用户选择保存的译文初稿，系统创建一个新的译文记录类型,进行保存 即：
```json
    "type": "draft",
    "text": "在新华社，中国日报，...",
    "timestamp": 1766103273668
```
